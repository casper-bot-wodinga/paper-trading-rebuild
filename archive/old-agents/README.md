# Old Agents — Archived

This directory contains the original standalone trader agent scripts
(`kairos.py`, `aldridge.py`, `stonks.py`) that were replaced by the
rebuild's agent workspace structure under `agents/trader-*/`.

## Why Archived

The old Python-only agent scripts have been superseded by the paper-trading
rebuild's multi-file workspace layout with separate AGENTS.md, SOUL.md,
TOOLS.md files for each trader. The new system supports richer agent identity,
skills integration, and easier configuration management.
